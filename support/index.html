2. Getting Started

Toggler is a mobile companion for managing feature groups, feature values, and rollout strategies on a FeatureHub-compatible server.

  1. Open Toggler.
  2. Enter your FeatureHub server URL.
  3. Enter your bearer token.
  4. Tap Connect.
  5. Select a project to view and manage feature groups.
Production use requires access to a FeatureHub-compatible backend and a valid bearer token.

3. App Review Demo

For App Store review or quick testing, Toggler includes an in-memory demo mode that does not call an external backend.

  1. Open the app.
  2. If onboarding appears, tap Get started or Skip.
  3. In Server URL enter demo.
  4. In Bearer token enter token.
  5. Tap Connect.

The demo project lets reviewers search groups, switch between list and grid views, edit a group, manage rollout strategies, duplicate a group, and test snapshot actions.

4. Troubleshooting

  • Check that your server URL is reachable from the device.
  • Confirm that your bearer token is valid and has access to the selected project.
  • Verify network, VPN, or firewall settings if projects or groups do not load.
  • Update to the latest version of Toggler when available.
  • If a problem continues, contact support with steps to reproduce it.

5. Privacy

Toggler is designed to minimize data collection. It does not include analytics, advertising, or tracking SDKs.

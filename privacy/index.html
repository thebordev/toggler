Privacy Policy for Toggler

This Privacy Policy explains how the Toggler mobile application handles user information. Toggler is designed to minimize data collection and keep operational FeatureHub data under the user's control.

1. Information We Collect

Toggler does not collect, sell, share, or use personal information for tracking or advertising.

The App:

  • does not require a Toggler account
  • does not collect analytics data
  • does not include advertising or tracking SDKs
  • does not collect location information
  • does not access contacts, camera, microphone, or photos

2. Information Stored on Device

To provide its core functionality, Toggler may store certain information locally on the user's device.

This may include:

  • the server URL entered by the user
  • the bearer token entered by the user
  • the selected app language
  • the onboarding completion state
  • local feature group snapshots created by the user
This information remains on the user's device unless the user sends data to their own FeatureHub-compatible server through the App's normal functionality.

3. Server Connections

Toggler connects only to the server URL provided by the user. The App uses the bearer token entered by the user to request projects, feature groups, feature values, and rollout strategy data from that server.

Production feature data is exchanged between the user's device and the server selected by the user. The developer of Toggler does not receive or process that data unless the user chooses a server operated by the developer.

4. Device and Technical Information

While Toggler itself does not collect analytics or tracking data, certain technical information may be processed automatically by Apple or application distribution platforms to provide installation, update, diagnostics, and security functionality.

This may include:

  • device model
  • operating system version
  • application version
  • basic crash or diagnostic information, if provided by the platform
Platform-provided information is handled according to the privacy practices of the applicable platform provider.

5. Permissions

Toggler does not request access to sensitive device permissions, including but not limited to:

  • Location
  • Camera
  • Microphone
  • Contacts
  • Photos
  • Phone state

6. Third-Party Services

Toggler does not integrate third-party SDKs or services that collect user data, such as:

  • analytics services
  • advertising networks
  • tracking tools
  • external user profiling services

7. Data Sharing

Toggler does not collect personal data for the developer. As a result, the developer does not share user personal information with third parties.

8. Data Retention

Locally stored information remains on the user's device until the user changes it, removes the App, clears local storage, or deletes local snapshots.

Data stored on a user's FeatureHub-compatible server is governed by that server's own retention and privacy practices.

9. Children’s Privacy

Toggler is intended for developer, release management, and business workflows. It is not directed to children.

The App does not knowingly collect personal information from children.

10. Security

Toggler uses standard platform networking to connect to user-provided servers. Users should protect their bearer tokens and only connect to trusted servers.

We take reasonable steps to develop the App using standard security practices, but no method of transmission or local storage is completely secure.

11. Changes to This Privacy Policy

We may update this Privacy Policy from time to time.

Any updates will be published on this page with an updated Last updated date.
